# QR Code Generator (Python)

A simple Python mini project that generates a QR code from a user-provided URL and saves it as an image file on the device.

## Features
- Generate QR codes from any URL
- Automatically saves the QR code as a PNG image
- Simple and beginner-friendly Python script

Make sure Python is installed, then install the required library:
